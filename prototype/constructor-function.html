What is 1. A constructor is a special function that creates and initializes an object instance of a class
2. In JavaScript, a constructor gets called when an object is created using the new keyword.

3. JavaScript এর সমস্ত ফাংশনই বাই ডিফল্ট constructor ফাংশন।
Example
function Person(fname, lname){
  this.fname = fname;
  this.lname = lname;
}

const person1 = new Person('Jhon', 'Doe')
const person2 = new Person('David', 'Luiz')

console.log(person1.fname, person1.lname)
console.log(person2.fname, person2.lname)
Output
Jhon Doe
David Luiz

  • We can create multiple objects using constructor function.
