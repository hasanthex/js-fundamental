What is ? Every Variable has a data type that tells what kind of data is being stored in a variable.
How many types? There are two types of data types in javascript
  1. Primitive Data Types
  2. Non-primitive Data Types
Primitive Data Types The predefined data types provided by JS are knows as Primitive data types its also known as in-build data types.
  1. Number
  2. String
  3. Undefined
  4. Boolean
  5. Null
Non-Primitive Data Types The data types that are derived from primitive data types known as non-primitive data types its also known as derived data types.
  1. Object
  2. Array
